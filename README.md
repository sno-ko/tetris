# tetris
### 遺伝的アルゴリズムを用いたテトリスAI（開発：2018年12~2019年4月）

実行方法：
`ruby tetris.rb`

Rubyバージョン：ruby 2.3.3

必要ライブラリ：DXRuby

操作方法：<br>
A，D：左移動，右移動 <br>
S：高速落下 <br>
W：ハードドロップ <br>
F，G：左回転，右回転 <br>
T：ホールド
